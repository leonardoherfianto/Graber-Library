Graber Canvas

you can use graber canvas to draw

To create canvas element you can use new Canvas(width,height,parent)

By default width and height canvas allways match to inner size of screen.

By default parent will allways set to document.body

Lets create our canvas

Code Sample new Canvas(w,h,p)
var parent = _G('#live_result');
mycanvas = new Canvas(400,400,parent[0]);

Canvas fill background

to fill the background you can use .fillBg(color)

in Graber Canvas, argument color present in 3 ways.

First as string like HEX ('#fff') and RGBA ('rgba(255,255,255,1)')

or as number like (255,255,255,1)

or as array like ([255,255,255,1])

Code Sample Canvas.fillBg(color)
mycanvas.fillBg('#fff');

Canvas Rectangle

Canvas Rectangle need 4 argument to create

with .rect(x,y,width,height)

Code Sample Canvas.rect(x,y,w,h)
mycanvas.begin(); /*same as canvas.context.beginPath() */
mycanvas.rect(100,100,200,200);
mycanvas.close(); /* same as canvas.context.closePath() */
mycanvas.fill(100); /* color will set to static value rgba(100,100,100,1) */

Canvas Clear

on Graber Canvas to clear the canvas you can use .clear()

Code Sample Canvas.clear()
mycanvas.clear();

Canvas Arc

on Graber Canvas to create a circle use .arc(x,y,radius,start,end,type)

x and y is the center arc. radius is size of the arc

start and end is the arc length from 0 - 1 in Graber

type is the shape of arc 'chord','pie','open'

Code Sample Canvas.arc(x,y,radius,start,end,type)
mycanvas.save(); /* same as canvas.context.save() */
mycanvas.fillBg('rgba(255,255,255,1)'); /* use string rgba as color fill */

mycanvas.arc(50,50,40); /* Full shape arc */
mycanvas.fill([255,0,0,1]); /* use array as color will result red color */

mycanvas.arc(150,50,40,0,0.5); /* Half shape arc */
mycanvas.fill(0,255,0,1); /* use number value as color will result green color */

mycanvas.arc(250,50,40,0,0.8); /* Chord shape arc */
mycanvas.fill('skyblue'); /* use color name as value*/

mycanvas.arc(50,150,40,0,0.2,'pie'); /* Pie shape arc */
mycanvas.fill('#ffff00'); /* use hex color as value */

mycanvas.arc(150,150,40,0,0.7,'open'); /* Open shape arc */
mycanvas.lineStyle(4); /* Same as canvas.context.lineWidth */
mycanvas.stroke('#ff00ff'); /* use hex color as value */

mycanvas.restore(); /* same as canvas.context.restore() */

Canvas Line

on Graber Canvas line is use with vector object.

Graber has Vector object

to create line in canvas use .line(vectorA,vectorB);

Code Sample Canvas.line(vectorA,vectorB)
mycanvas.clear();
mycanvas.fillBg(255);

var vecA = new Vector(10,10); /* create vector at x and y position */
var vecB = new Vector(300,240); /* create vector at x and y position */
mycanvas.save();
mycanvas.begin();
mycanvas.line(vecA,vecB);
mycanvas.close();
mycanvas.stroke(170);
mycanvas.restore();

Canvas Fill Text

To insert the text will fill in color to canvas use .fillText(text,x,y,option)

x and y is the post to insert the text

on option you can set the font-family,font-size and textAlign

Code Sample Canvas.fillText(text,x,y,option)
mycanvas.clear();
mycanvas.fillBg(255);

mycanvas.save();
mycanvas.fillStyle('red'); /* set fill color to red */
mycanvas.fillText('ALPHA',100,100, {
'font':'20px Arial', 'textAlign':'center', });
mycanvas.restore();

Canvas Shadow

To set shadow effect in canvas, you can use .shadow(x_offset,y_offset,blur,color)

x and y offset is the distance between shadow and canvas object.

Code Sample Canvas.shadow(x_offset,y_offset,blur,color)
mycanvas.clear();
mycanvas.fillBg(255);
mycanvas.save();
mycanvas.shadow(3,3,5,'rgba(0,0,0,1)');
mycanvas.polygon(200,200,80,6);
mycanvas.fill(100);
mycanvas.restore();

Graber Animator

Graber also has animator engine called Anime. Combine with canvas we can create our animation

To start we need to place our function in Anime.onrender.your_function_name = function(){/ /your_code; }

After that to run the animation, we use Anime.start()

Code Sample Anime.onrender & Anime.start()
mycanvas.clear();
mycanvas.fillBg(255);
/* create something to animation */
ball = {
pos : new Vector(mycanvas.w/2,30), vel : new Vector() };
gravity = new Vector(0,1);
/* set animation render */
Anime.onrender.myAnimation = function(){
mycanvas.clear(); mycanvas.fillBg(255); if(ball.pos.y+15 >= mycanvas.h){ ball.vel = ball.vel.mult(-0.99); ball.pos.y = mycanvas.h - 15; }; ball.vel.add(gravity); ball.pos.add(ball.vel); mycanvas.begin(); mycanvas.arc(ball.pos.x,ball.pos.y,15); mycanvas.fill('coral'); mycanvas.close(); };
/* start animation */
Anime.start();

Stop Animation

To stop the animation you can use .break()

And to add a even listener when the animation has stop you can use .onbreak.your_function_name = function(){ //your_code; }

To try our code below let run again our last animation example and stop the animation.

Code Sample Anime.break() & Anime.onbreak
/* Click try to stop the animation*/
Anime.onbreak.myStopFunction = function(){
alert('Anime has stop!'); };
Anime.break();
