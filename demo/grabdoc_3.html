Graber Array and Number

In Graber we provide some additional library for help manage Array and Number

For listing Array with Graber you can use Array.sortAsc() and Array.sortDsc()

What the different ? In Graber Sorting Array you don't need convert string number into number object

Code Sample Array.sortAsc() & Array.sortDsc()
var myArray = ['Lisa','Rudi','Bart','Angela','Murphy'];
var myStringNumber = ['9','20','3','1','10'];

_G('#live_result').add('P').text('Sort Ascending = ' + myArray.sortAsc().join(',') );
_G('#live_result').add('P').text('Sort Descending = ' + myStringNumber.sortDsc().join(',') );

Array max and min

To get max number from Array, you can use Array.max()

And to get min number from Array, you can use Array.min()

Code Sample Array.max() & Array.min()
var myArray = [53,23,65,43,12,60,71];
var myStringNumber = ['9','20','3','1','10'];

_G('#live_result').add('P').text('Max number in myArray = ' + myArray.max() );
_G('#live_result').add('P').text('Min number in myStringNumber = ' + myStringNumber.min() );

Array near and far

Array near and far is to compare some number with array and return the closest/futherest number in array

Code Sample Array.near(number) & Array.far(number)
var myArray = [53,23,65,43,12,60,71];
_G('#live_result').add('P').text('The closest myArray to 61 = ' + myArray.near(61) );
_G('#live_result').add('P').text('The futherest myArray to 61 = ' + myArray.min(61) );

Array Sum

To sum an array and get the result use Array.sum()

Code Sample Array.sum()
var myArray = [53,23,65,43,12,60,71];
_G('#live_result').add('P').text('The sum from myArray = ' + myArray.sum() );

Graber Scaler

To scale the number from compare two order number and return the scale number, you can use Number.scaler(min,max,comparemin,comparemax)

min and max is the limit of your number.

comparemin and comparemax is the compare limit number to scale your number.

Code Sample Number.scaler(min,max,comparemax,comparemin)
var mynumber = 450;
var resultNum = mynumber.scaler(0,window.innerHeight,0,360);
var reverseResultNum = mynumber.scaler(0,window.innerHeight,360,0);

_G('#live_result').add('P').text('let scaler our number (450) from 0-window.innerHeight to 0-360 degree = ' + resultNum + ' degree');
_G('#live_result').add('P').text('let scaler in reverse way our number (450) from 0-window.innerHeight to 360-0 degree = ' + reverseResultNum + ' degree');

Graber Fixed

Graber Fixed same as toFixed function but the different is Fixed return number not string

Code Sample Number.fixed(decimal)
var mynumber = 450.429163982;
var resultNum = mynumber.fixed(2);
_G('#live_result').add('P').text('from 450.429163982 we fix to = ' + resultNum);

Graber Radians

Graber Radians is for convert some number (in degree) to radians number

Code Sample Number.rad()
var mydegree = 30;
var myradians = mydegree.rad();
_G('#live_result').add('P').text('from 30 degree we convert to radians = ' + myradians);

Graber Degree

Graber Degree is for convert some number (in radians) to degree number

Code Sample Number.deg()
var myradians = 0.5235987755982988;
var mydegree = myradians.deg();
_G('#live_result').add('P').text('from 0.5235987755982988 in radians we convert to degree = ' + mydegree.fixed(0));

Graber Viewport

Graber also has way to convert pixel to Viewport size for css style

To convert from pixel to viewport width you can use .vw(screenWidth|null)

To convert from pixel to viewport height you can use .vh(screenHeight|null)

By default screenWidth and screenHeight is set to innerSize of the screen

Code Sample .vw(screenWidth|null) & .vh(screenHeight|null)
var mywidth = 473; /* in pixel */
var myheight = '240px'; /* from collected css */

_G('#live_result').add('P').text('473px in viewport width = ' + mywidth.vw() + 'vw');
_G('#live_result').add('P').text('240px in viewport height = ' + myheight.vh() + 'vh');

Graber Vector

Graber vector is object number in 2 dimensional cartesius number

To create a vector, you can use new Vector(numX,numY);

Code Sample new Vector(numX,numY)
var myvector = new Vector(20,80);

var parent = _G('#live_result');
mycanvas = new Canvas(200,200,parent[0]);
mycanvas.fillBg(50);
mycanvas.save();
mycanvas.arc(myvector.x,myvector.y,2);
mycanvas.fill(255);
mycanvas.restore();

Vector copy, set, and clone

Vector .copy(other_vector) will copy value from other vector and set as self value.

Vector .set(x,y) is for set vector based provide x and y value

Vector .clone() will copy value from current vector and return it as new vector.

Code Sample .copy(other_vector), .set(x,y) & .clone()
var myvector = new Vector(20,80);
var otherVector = new Vector(200,500);

_G('#live_result').add('P').text('clone myvector = '+JSON.stringify( myvector.clone() ));
_G('#live_result').add('P').text('set myvector = '+JSON.stringify( myvector.set(100,400) ));
_G('#live_result').add('P').text('myvector copy from otherVector = '+JSON.stringify( myvector.copy(otherVector) ));

Vector multiple and divide

Vector .mult(x,y|null) is for multiple the vector with number.

Vector .div(x,y|null) is for divide the vector with number.

Code Sample .mult(x,y|null) & .div(x,y|null)
var myvector1 = new Vector(2,4);
var myvector2 = new Vector(12,18);

_G('#live_result').add('P').text('myvector1 : '+ JSON.stringify(myvector1) +' * 5 = '+JSON.stringify( myvector1.mult(5) ));
_G('#live_result').add('P').text('myvector2 : '+ JSON.stringify(myvector2) +' / (6,9) = '+JSON.stringify( myvector2.div(6,9) ));

Vector adding and substract

Vector .add(otherVector) is for adding the vector with other vector.

Vector .sub(otherVector) is for substract the vector with other vector.

Code Sample .mult(x,y|null) & .div(x,y|null)
var myvector1 = new Vector(2,4);
var myvector2 = new Vector(12,18);
var otherVector = new Vector(5,10);

_G('#live_result').add('P').text('myvector1 : '+ JSON.stringify(myvector1) +' + otherVector = '+JSON.stringify( myvector1.add(otherVector) ));
_G('#live_result').add('P').text('myvector2 : '+ JSON.stringify(myvector2) +' - otherVector = '+JSON.stringify( myvector2.sub(otherVector) ));

Vector Distance & Distance Square

Vector .dist(otherVector) will return distance length from delta : vector to other vector.

Vector .magSq() will return the length square area of the vector to origin or delta vector

Vector .mag() will return the distance length of vector to origin or delta vector.

Code Sample .dist(otherVector), .magSq() & .mag()
var myvector1 = new Vector(20,14);
var myvector2 = new Vector(40,78);
var dist = myvector1.dist(myvector2);

mycanvas.clear();
mycanvas.fillBg(50);
mycanvas.save();
mycanvas.begin();
mycanvas.line(myvector1,myvector2);
mycanvas.close();
mycanvas.stroke(255);
mycanvas.begin();
mycanvas.rect(0,0,myvector1.x,myvector1.y);
mycanvas.close();
mycanvas.stroke('red');
mycanvas.begin();
mycanvas.line(new Vector(),myvector2);
mycanvas.close();
mycanvas.stroke('red');
mycanvas.restore();
_G('#live_result').add('P').text('distance from myvector1 to myvector2 = '+JSON.stringify( dist ));
_G('#live_result').add('P').text('length square area from myvector1 to point (0,0) = '+JSON.stringify( myvector1.magSq() ));
_G('#live_result').add('P').text('distance from myvector2 to point (0,0) = '+JSON.stringify( myvector2.mag() ));

Vector Direction, Angle, Dot Produce

Vector .dir() will return direction (in radians) from vector to origin or delta vector

Vector .angle(otherVector) will return the angle (in radians) from vector to other vector

Vector .dot(otherVector) will return the dot produce of vector and otherVector

Code Sample .dir(), .angle(otherVector) & .dot(otherVector)
var myvector1 = new Vector(2,4);
var myvector2 = new Vector(12,18);
var angle = myvector1.angle(myvector2);
var dir = myvector1.dir();
var dot = myvector1.dot(myvector2);

_G('#live_result').add('P').text('angle from myvector1 to myvector2 = '+JSON.stringify( angle ));
_G('#live_result').add('P').text('direction from myvector1 to point (0,0) = '+JSON.stringify( dir ));
_G('#live_result').add('P').text('dot produce from myvector1 and myvector2 = '+JSON.stringify( dot ));

Vector Rotate, Heading, Reverse

Vector .rotate(radians) will adding current vector angle with set rotate and return new vector by the direction.

Vector .heading(radians) will change without adding the vector angle to set angle.

Vector .reverse() will reverse the vector value from x,y to -x,-y

Code Sample .rotate(radians), .heading(radians) & .reverse()
var myvector1 = new Vector(2,4);
var myvector2 = new Vector(2,4);

_G('#live_result').add('P').text('new angle for myvector1 = '+JSON.stringify( myvector1.rotate(0.5235987755982988) ));
_G('#live_result').add('P').text('new direction for myvector2 = '+JSON.stringify( myvector2.heading(0.5235987755982988) ));
_G('#live_result').add('P').text('reverse value from myvector1 '+JSON.stringify( myvector1.reverse() ));

Vector Normalize, Limit, Project

Vector .norm() will return the normal value of vector between 0-1.

Vector .limit(max) will limit the vector to set value by square area.

Vector .project(other,origin) will return vector projection from other vector by origin point

Code Sample .rotate(radians), .heading(radians) & .reverse()
var myvector1 = new Vector(2,4);
var myvector2 = new Vector(6,10);
var myvector3 = new Vector(4,12);

_G('#live_result').add('P').text('normalize myvector1 = '+JSON.stringify( myvector1.norm() ));
_G('#live_result').add('P').text('limit myvector2 = '+JSON.stringify( myvector2.limit(3) ));
_G('#live_result').add('P').text('projection myvector3 with myvector2 by origin myvector1 = '+JSON.stringify( myvector3.project(myvector2,myvector1) ));
