Wellcome to Graber documentation

On this section we will Introduce the basic Graber Library Collections for manage the DOM Element

Below we have 2 panels. The left panel it's sample code of Graber Library and the right panel it's live result

Create and Append DOM Element

With Graber you can create html DOM element.

On sample below we make an UL tag with class name listbox,

and continue with create 6 LI tag with class name list and fill with text "created list".

Then append to element with id live_result

Code Sample Graber.add()
_G('#live_result').add('UL.listbox').add('LI.list*5').text("created list");

Insert Before and After

If need to insert element before and after you can use Graber selector

on first sample below we create new LI and insert before UL children index 3

on second sample below we create new LI and insert after UL children index 4 or before index 5

Code Sample Graber.add('element'+'(insert after/before)')
var firstSample = _G('.listbox').add('LI.list(3)').text('new list insert before');

var secondSample = _G('.listbox').add('LI.list(5)').text('new list insert after');

Adding CSS Style

You can add stylelist css on DOM element after or when the element is create

with .css({'style' : 'value'}) you can add the css style as much you want.

Code Sample Graber.css({})
_G('.listbox').css({
'list-style-type':'none', 'width' : '300px' }).child().css({
'border':'1px solid rgba(0,0,0,0.09)', 'padding':'5px 6px' });

Select Element

If you want to select some element in children or from Graber HTML collect element,

you can use .select(start_index, Increasement, end_index | null);

On sample below we will select some element from our list element and create stripted style.

Code Sample Graber.select(start_index, Increasement, end_index | null)
_G('.list').select(0,2).css({
'background':'rgb(111, 58, 58)', 'color':'rgb(232, 206, 71)' })

Loop

If you want do loop in Graber html collection element,

you can use .map(function(item,index){});

On sample below we will grab all LI with class name list and fill with some data.

Code Sample Graber.map(function(item,index){ //your code })
var mydata = ['ALPHA','BRAVO','CHARLIE','DELTA','ECHO','FOXTROT','GOLF'];
_G('.list').map(function(item,index){
item._G().text(mydata[index]); });

Get Style value

On Graber when you want to get the style value and it not exist in property style,

Graber will return value from Computed Style. You can use .css('style-name')

On sample below we will get LI style height when it not exist or unset.

Code Sample Graber.css('style-name')
var styleHeight = _G('.list').css('height');

_G('#live_result').add('P').text('height = '+styleHeight.join(','));

AddEventListener

On Graber you can also add event listener,

with .on('event', callback_function);

On sample below we add event click on our list, go ahead to try click

Code Sample Graber.on('event', callback_function)
_G('.list').on('click',function(e){
alert(this.innerHTML); console.log(e) })

RemoveEventListener

On Graber you can also remove event listener, with .off('event', function_name)

if you just want remove the last event you added you just use .off()

On sample above we just added the click event so to remove you can see example below

Code Sample Graber.off('event', function_name)
_G('.list').off(); //remove the last added event.

Set and Get Property Element

On Graber you can set property element with .prop({'name':'value'});

To get the property value you can use .prop('prop-name')

some rule is attach in this method where set must in object and get in string

On sample below we set prop label to the list element and get the prop tagname

Code Sample Graber.prop({'name':'value'}) or Graber.prop('prop-name')
var list_tagname = _G('.list').prop({'label':'myLabel'}).prop('tagName');

_G('#live_result').add('P').text('tagname = '+list_tagname.join(', '));

Set and Get Attribute Element

On Graber you can set attribute element with .attr({'name':'value'});

To get the property value you can use .attr('attr-name')

some rule is attach in this method where set must in object and get in string

On sample below we set attr title to the list element and get attr we create

Code Sample Graber.attr({'name':'value'}) or Graber.attr('attr-name')
_G('.list').attr({'title':'myTitle'});

_G('#live_result').add('P').text('Attribute Title = ' + _G('.list').attr('title').join(', '));

innerHTML and innerText

On Graber you can set or get innerHTML with .html('your DOM') or just .html() to get.

To insert or get plain text in Graber you can use .text('your text') or just .text() to get

Code Sample Graber.html('your DOM'|null) or Graber.text('your text'|null)
_G('#live_result').add('P').text('Hello'); /* add text to P element. */

_G('#live_result').add('P').html('<b>Hello</b>'); /* add innerHTML to P element. */

BoundingClientRect and Offset Element

On Graber you can get BoundingClientRect with .bound();

And get Offset Element with .offset();

On sample below we will get BoundingClientRect and Offset Element from UL

Code Sample Graber.bound() and Graber.offset()
var bound = _G('.listbox').bound(); var offset = _G('.listbox').offset(); _G('#live_result').add('P').text('BoundingClientRect = ' + JSON.stringify(bound[0])); _G('#live_result').add('P').text('Offset Element = ' + JSON.stringify(offset[0]));

Add,Remove and Has Class

On Graber to add a new class in element use .addClass(className);

To check is the element already has the class use .hasClass(className);

To remove the class from element use .removeClass(className);

Code Sample Graber.addClass(), Graber.hasClass(), Graber.removeClass()
var ul = _G('.listbox').addClass('myNewClass');
if(ul.hasClass('myNewClass')[0]){
_G('#live_result').add('P').text('Already has myNewClass as className'); ul.removeClass('listbox'); _G('#live_result').add('P').text(ul.prop('className').join('')); };

Parent,Child and Siblings Element

On Graber you can get a parent,child, and siblings in simple way

use .parent() to get parent element and .child() to get child element.

use .siblings() to get all siblings element

Code Sample Graber.parent(), Graber.child(), Graber.siblings()
console.log( _G('.list').parent() );

console.log( _G('UL').child() );

console.log( _G('UL').siblings() );

Prev and Next Element Siblings

On Graber to get next element sibling use .next()

and to get previous element sibling use .prev()

In sample below we will get list index 3 and get prev sibling then

we get the next sibling from index 3

Code Sample Graber.prev() and Graber.next()
var list = _G('.list');
var prev = list[3]._G().prev();
var next = list[3]._G().next();
_G('#live_result').add('P').text('my prev sibling innerText is : ' + prev.text() + ' and my innerText is : '+list[3]._G().text());
_G('#live_result').add('P').text('and my next sibling innerText is : ' + next.text());
